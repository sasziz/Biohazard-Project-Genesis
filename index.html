Old Liminal pack is no longer used

The server changed to Biohazard: Project Genesis. Delete or ignore the old Liminal Industries instance and install the new pack from this page.

Before you start

  • Use Minecraft 1.20.1, Forge 47.4.0, and Java 17.
  • Use the exact Biohazard modpack zip from the download page.
  • Do not add random extra mods.
  • Offline users should keep the same username or inventory/progress can look reset.

1. Download the modpack

  1. Click Download Modpack above.
  2. On the GitHub release page, download the Biohazard modpack .zip from Assets.
  3. Keep it as a zip file. Most launchers import it directly.

If the release page shows multiple files, download the newest Biohazard client/modpack zip, not the old Liminal zip.

2. Official Minecraft account setup

Prism Launcher

  1. Open Prism Launcher.
  2. Click Add Instance.
  3. Choose Import.
  4. Select the downloaded Biohazard modpack zip.
  5. Wait for Prism to install/download everything.
  6. Launch the instance.
  7. Add the server address below.

CurseForge App

  1. Open CurseForge App.
  2. Go to Minecraft.
  3. Click Import or Import Profile.
  4. Select the downloaded Biohazard modpack zip.
  5. Wait for the profile to be created.
  6. Press Play.
  7. Add the server address below.

3. Offline / unofficial setup — SKlauncher recommended

For offline or alternative launcher users, SKlauncher is recommended.

Try this first: import the zip directly

  1. Open SKlauncher.
  2. Go to the modpacks / instances / installations area.
  3. Choose the option to import or add a modpack.
  4. Select the downloaded Biohazard modpack zip.
  5. Let SKlauncher install everything.
  6. Launch the modpack.
  7. Add the server address below.

If SKlauncher import fails

Install/import the pack in Prism Launcher or CurseForge first, launch it once, then copy the completed instance files into a SKlauncher Forge 1.20.1 / 47.4.0 installation.

Voice chat works. Press V in-game. Make sure Minecraft has microphone permission.

5. Offline skins

After joining, use:

/skin set mojang Username

Example:

/skin set mojang Notch

If it does not update:

/skin refresh

Or leave and rejoin.

6. Common fixes

Missing mods / mod mismatch

You are probably using the old Liminal instance or the wrong Biohazard version. Delete the instance and import the newest zip again.

Game crashes before opening

Use Java 17, Minecraft 1.20.1, and Forge 47.4.0.

SKlauncher import failed

Use the fallback copy method above.

Bad FPS

Lower render distance, turn off shaders, close background apps, and allocate more RAM if your PC has enough.

Voice chat does not work

Press V, check microphone input, and allow microphone permission in Windows privacy settings.
