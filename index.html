Before you start

  • You need the exact same modpack version as the server.
  • Do not add random extra mods.
  • Use Minecraft 1.20.1, Forge 47.4.13, and Java 17.
  • Offline users should not keep changing usernames or their inventory/progress can look reset.

1. Download the modpack

  1. Click Download Modpack above.
  2. Download Liminal Industries-1.19.3(1).zip.
  3. The file is about 64.9 MB.
  4. Keep it as a zip. Most launchers import it directly.

2. Official Minecraft account setup

Prism Launcher

  1. Open Prism Launcher.
  2. Click Add Instance.
  3. Choose Import.
  4. Select Liminal Industries-1.19.3(1).zip.
  5. Wait for it to install/download everything.
  6. Launch the instance.
  7. Add the server address below.

CurseForge App

  1. Open CurseForge App.
  2. Go to Minecraft.
  3. Click Import or Import Profile.
  4. Select Liminal Industries-1.19.3(1).zip.
  5. Wait for it to create the profile.
  6. Press Play.
  7. Add the server address below.

3. Offline / unofficial setup - SKlauncher recommended

For offline/alternative launcher users, SKlauncher is recommended.

Try this first: import the zip directly

  1. Open SKlauncher.
  2. Go to the modpacks/instances/installations area.
  3. Choose the option to import/add a modpack.
  4. Select Liminal Industries-1.19.3(1).zip.
  5. Let SKlauncher install everything.
  6. Launch the modpack.
  7. Add the server address below.

If SKlauncher import fails

Install/import the pack in Prism Launcher or CurseForge first, launch it once, then copy the completed instance files into a SKlauncher Forge 1.20.1 / 47.4.13 installation.

Fallback copy method

  1. Import the zip in Prism or CurseForge.
  2. Launch it once so all mods/configs download.
  3. Open that installed modpack folder.
  4. In SKlauncher, create a Minecraft 1.20.1 Forge 47.4.13 installation.
  5. Open the SKlauncher installation folder.
  6. Copy folders like mods, config, defaultconfigs, kubejs, resourcepacks, and shaderpacks if they exist.
  7. Launch from SKlauncher.

Or leave and rejoin.

6. Common fixes

“Incompatible FML modded server” but I can still join

Ignore it if you can join and play. It usually means the server has an extra server-side mod.

Missing mods / mod mismatch

Delete the instance and import the newest zip again.

SKlauncher import failed

Use the fallback copy method above.

Game crashes before opening

Use Java 17, Minecraft 1.20.1, and Forge 47.4.13.

Bad FPS

Lower render distance, turn off shaders, close background apps, and allocate more RAM if your PC has enough.
